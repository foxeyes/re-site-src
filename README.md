## re4ma site

### Installation:

`git clone git@github.com:uploadcare/experimental-blog.git ./`

Then:

`npm run prepare`

### Develop:

Start local server:

`npm run dev`

Publish:

`npm run pub`

### Project setup:

`./project.json`

### Dependency docs:

re4ma: https://github.com/uploadcare/re4ma#readme

symbiote: https://github.com/uploadcare/symbiote.js/tree/v2#readme
